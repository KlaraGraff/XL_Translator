# XL Translator 版本更新日志

## V2.2.0
- 新增更多内置目标语言支持。
- 新增任意语言翻译成中文的能力。

## V2.1.0
- 修复“公式生成的显示文本”在翻译后无法正确回填的问题。
- 新增可选项“公式文本按显示值回填”，默认开启。
- 开启后会按公式显示值匹配翻译并回填。
- 命中后对应单元格会改写为静态文本，不再保留原公式。

## V2.0.1
- 调整启动方式：第一次启动会显示初始化过程，初始化完成后，后续启动默认保持静默启动。
- 静默启动失败时，会自动切回可见启动方式。
- 更新了启动脚本、首次使用说明和分发相关说明。

## V2.0.0
- 新增统一的版本信息管理。
- 新增应用图标和侧边栏品牌显示。
- 分发包文件名开始带版本号，方便区分和保留历史版本。
- 打包时会一并带上应用图标和版本信息。
- 更新了分发流程和相关说明文档。

## V1.3.0
- 调整了翻译页的页面布局。
- 新增系统原生文件夹选择和 Excel 文件选择功能。
- 调整了术语库（TM）管理页的搜索、分页、删除确认和提示词设置相关功能。
- 增加了整应用和页面视觉检查相关的自测支持。

## V1.2.0
- 新增项目自测规范和自测工具。
- 目标语言选择从英文、法文两项，调整为统一的目标语言选择方式。
- 内置目标语言从英文、法文扩展为多种语言。
- 新增自定义目标语言功能。
- 新增最近使用目标语言记录功能。
- 翻译页和术语库（TM）页改为共用同一套目标语言处理逻辑。

## V1.0.0
- 提供基础应用结构。
- 提供 Excel 翻译功能。
- 提供术语库（TM）管理功能。
- 提供基础设置功能。
- 提供启动脚本和分发脚本。
- 初始支持英文和法文。
